🧘 Conversational AI Coach

An ICF-style multilingual conversational AI coaching assistant built using OpenAI, Whisper, TTS, and Gradio.

View GitHub Watch Demo

Project Overview

This project explores how conversational AI can simulate reflective coaching conversations using voice interaction, multilingual prompting, conversational memory, and structured coaching constraints.

Key Features

🎙️ Voice Interaction

Speech-to-text using Whisper and AI-generated voice responses using OpenAI TTS.

🌍 Multilingual

Supports English, Hindi, and Kannada conversational coaching.

🧠 Conversational Memory

Maintains session context across multi-turn coaching interactions.

🧭 Prompt Engineering

Designed to avoid advice and instead encourage reflective self-discovery.
