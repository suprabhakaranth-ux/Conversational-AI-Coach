🧘 Conversational AI Coach

A multilingual voice-enabled conversational AI coaching assistant designed to simulate reflective, ICF-style coaching conversations using speech recognition, conversational memory, prompt engineering, and AI voice interaction.

View GitHub Watch Demo

Why This Project

Most AI assistants are designed to answer questions, provide solutions, or automate tasks. Professional coaching works differently — it focuses on reflection, self-discovery, active listening, and open-ended questioning.

This project explores whether conversational AI can simulate a more reflective coaching experience through multilingual voice interaction, conversational continuity, and coaching-oriented prompt engineering.

Core Features

🎙️ Voice-Based Coaching

Users can speak naturally through microphone input and receive AI-generated spoken coaching responses.

✍️ Editable Transcript Layer

Speech recognition systems may occasionally misinterpret accents or pronunciation. The transcript is shown before submission so users can review and correct it if required.

🌍 Multilingual Support

Supports conversational coaching interactions in English, Hindi, and Kannada.

🧠 Conversational Continuity

Maintains session context across multiple coaching turns for a more natural conversational flow.

🧭 ICF-Style Coaching Prompts

Designed around reflective coaching principles using active listening, empathy, clarity, and self-discovery-oriented dialogue patterns.

👤 Coach Persona Customization

Coaching tone, empathy level, directness, warmth, questioning style, and communication behavior can be customized through prompt engineering.

Architecture

The application follows a layered conversational AI workflow where each component handles a specific part of the coaching interaction pipeline.

👂 Speech Recognition Layer

OpenAI Whisper

Converts multilingual voice input into text for downstream conversational processing.

✍️ Transcript Validation Layer

Editable Transcript Interface

Allows users to validate or correct transcripts before generating the coaching response.

🧠 Conversational Intelligence Layer

OpenAI GPT

Handles conversational reasoning, coaching continuity, and reflective question generation.

🧭 Coaching Persona Layer

Prompt Engineering System

Defines coaching tone, empathy, directness, conversational warmth, and coaching behavior constraints.

🗣️ Voice Synthesis Layer

OpenAI TTS

Converts generated coaching responses into natural spoken audio replies.

🖥️ Interaction Layer

Gradio Interface

Provides microphone input, audio playback, session state management, and rapid conversational AI prototyping.

☁️ Deployment Layer

Hugging Face Spaces

Hosts the conversational AI prototype in a lightweight deployment environment.

🐍 Application Layer

Python

Acts as the orchestration layer connecting voice, prompting, conversational logic, and AI services.

Privacy & Confidentiality

🔒 Transient Sessions

Coaching sessions are designed as temporary conversational interactions.

🧹 No Intentional Storage

Conversations are not intentionally stored after the session ends.

🔑 Protected API Keys

API credentials remain server-side protected and are never exposed to users.

Disclaimer

This prototype is intended as a reflective conversational coaching assistant and not as a therapist, mental health professional, or medical advisory system.

Fork This For Yourself

The architecture was intentionally designed to be modular and customizable for different conversational AI workflows and interaction styles.

👤 Customize the Coach Persona

Modify empathy level, coaching tone, directness, conversational warmth, questioning style, and coaching frameworks.

🌍 Add More Languages

Extend the multilingual prompting layer to support additional regional or global languages.

🎯 Change the Use Case

Adapt the workflow for mentoring assistants, interview coaching, journaling companions, leadership simulations, or learning tools.

🧠 Add Memory & Personalization

Extend the system using persistent memory, vector databases, session summaries, and personalized coaching profiles.

🔓 Replace the AI Models

Swap OpenAI services with local/open-source models, Ollama, Groq, Anthropic, Gemini, or other providers.

☁️ Deploy Anywhere

Deploy the workflow using Hugging Face Spaces, Streamlit Cloud, Render, AWS, Azure, or custom backend infrastructure.
